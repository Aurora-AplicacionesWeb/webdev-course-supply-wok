# Fundamentos de Desarrollo Web

## Resumen del Curso
Este curso de 1 hora y 15 minutos introduce a estudiantes de secundaria a la creación de sitios web sencillos con HTML y CSS. **¡No requiere descargas!** Solo abre tu navegador web.

* **Duración total**: ~75 minutos (8 módulos)
* **Público objetivo**: Estudiantes de 12 a 17 años sin experiencia en programación
* **Prerrequisitos**: Ninguno
* **Herramientas necesarias**: **¡Solo tu navegador web!** (Chrome, Firefox, Safari, Edge)
* **📂 Repositorio de código fuente**: [https://github.com/Aurora-AplicacionesWeb/webdev-course-supply-wok](https://github.com/Aurora-AplicacionesWeb/webdev-course-supply-wok)

---

## Secuencia de la lección

### Lección 1: Introducción al desarrollo web (7 minutos)
* **Descripción**: Aprende de qué están hechos los sitios web, el modelo cliente-servidor y la diferencia entre frontend y backend.
* **Enlace**: [Ver la lección](https://youtu.be/placeholder)
* **Consejos clave**: El código es solo texto interpretado por el navegador.
* **Empieza a programar**: [Open JSFiddle](https://jsfiddle.net/)
* 💻 **¡No necesitas registrarte!**

### Lección 2: Estructura HTML básica (10 minutos)
* **Descripción**: Conoce qué es una etiqueta, sus atributos y el esqueleto obligatorio de una página web.
* **Enlace**: [Ver la lección](https://youtu.be/placeholder)
* **Consejos clave**: Toda página requiere `<!DOCTYPE html>`, `<html>`, `<head>`, y `<body>`.
* **Práctica**: [Clic para programar](https://jsfiddle.net/)
* 🚀 **¡Comienza al instante!**

### Lección 3: Elementos HTML comunes (12 minutos)
* **Descripción**: Agrega encabezados, párrafos, listas, imágenes y enlaces para hacer tu página interactiva.
* **Enlace**: [Ver la lección](https://youtu.be/q_7JTrIDvN0)
* **Consejos clave**: Usa las etiquetas `<h1>` a `<h6>`, `<p>`, `<ul>`, `<ol>`, `<img>`, `<a>`.
* **Práctica**: [Clic para programar](https://jsfiddle.net/)
* 💻 **¡No necesitas registrarte!**

### Lección 4: Introducción a CSS (10 minutos)
* **Descripción**: Aprende qué es CSS y cómo darle estilo a tu HTML vinculando un archivo externo.
* **Enlace**: [Ver la lección](https://youtu.be/UCWVvuEmTBQ)
* **Consejos clave**: Vincula estilos con `<link>`, usa selectores por etiqueta, clase (`.clase`) e ID (`#id`).
* **Práctica**: [Aplicar estilo con un clic](https://jsfiddle.net/)
* 🚀 **¡Comienza al instante!**

### Lección 5: Estilo simple (10 minutos)
* **Descripción**: Manipula colores de texto y fondo, fuentes y el modelo de caja express para ordenar tus contenidos.
* **Enlace**: [Ver la lección](https://youtu.be/0Z6JdahAulw)
* **Consejos clave**: Utiliza `color`, `background-color`, `font-family` y `margin: 0 auto` para centrar contenedores.
* **Práctica**: [Clic para programar](https://jsfiddle.net/vbshpmLd/5/)
* 💻 **¡No necesitas registrarte!**

### Lección 6: Creación de una página web básica - HTML (10 minutos)
* **Descripción**: Comienza el proyecto final maquetando la estructura de tu tarjeta de presentación (Profile Card).
* **Enlace**: [Ver la lección](https://youtu.be/placeholder)
* **Consejos clave**: Mantén el HTML bien anidado, estructurado semánticamente y limpio.
* **Práctica**: [Crea tu Perfil (HTML)](https://jsfiddle.net/)
* 🚀 **¡Comienza al instante!**

### Lección 7: Creación de una página web básica - CSS (10 minutos)
* **Descripción**: Estiliza la tarjeta de presentación aplicando colores, bordes redondeados y márgenes internos.
* **Enlace**: [Ver la lección](https://youtu.be/placeholder)
* **Consejos clave**: Usa `border-radius`, `padding`, y limita el ancho máximo del contenedor.
* **Práctica**: [Crea tu Perfil (CSS)](https://jsfiddle.net/)
* 💻 **¡No necesitas registrarte!**

### Lección 8: Recomendaciones y errores comunes (6 minutos)
* **Descripción**: Descubre las mejores prácticas de indentación, nombres de archivos en minúscula y los errores más comunes de sintaxis.
* **Enlace**: [Ver Lección](https://youtu.be/placeholder)
* **Consejos Clave**: Valida siempre las etiquetas de cierre, vinculación de archivos y la Documentación Web de MDN.
* **Comparte tu Trabajo**: [Envía tu Creación](https://forms.gle/your-form)
* 🚀 **¡Guarda y comparte!**
