# Curso Relámpago: Fundamentos de Desarrollo Web
## Estructura de 8 Módulos (Duración Total: 75 Minutos)
---

## Clase 1: Introducción al desarrollo web (7 minutos)
**Qué abarca:**
* Concepto básico: ¿Qué es un sitio web y cómo funciona el modelo cliente-servidor?
* Diferencia rápida entre frontend (lo visual) y backend (los datos).
* Las herramientas: El navegador y un editor de texto (mención rápida a VS Code).
* **Meta:** Que entiendan que el código es solo texto interpretado por el navegador.
* **Enlaces:** [C# Playground (DotNetFiddle)](https://dotnetfiddle.net/) | [Material de la Clase](starter-files/lesson-1-starter.md) ([Completado](completed-examples/lesson-1-completed.md), [Scripts](scripts/lesson-1-scripts.md))

## Clase 2: Estructura HTML básica (10 minutos)
**Qué abarca:**
* ¿Qué es una etiqueta? (Apertura, contenido, cierre).
* Atributos básicos (`nombre="valor"`).
* El esqueleto obligatorio de toda página web: `<!DOCTYPE html>`, `<html>`, `<head>` (metadatos) y `<body>` (lo visible).
* **Demostración:** Escribir el esqueleto desde cero en vivo.
* **Enlaces:** [C# Playground (DotNetFiddle)](https://dotnetfiddle.net/) | [Material de la Clase](starter-files/lesson-2-starter.md) ([Completado](completed-examples/lesson-2-completed.md), [Scripts](scripts/lesson-2-scripts.md))

## Clase 3: Elementos HTML comunes (12 minutos)
**Qué abarca:**
* Jerarquía de textos: Encabezados (`<h1>` a `<h6>`) y Párrafos (`<p>`).
* Listas: Desordenadas (`<ul>`) y Ordenadas (`<ol>`) con sus ítems (`<li>`).
* Medios y enlaces: Imágenes (`<img src="" alt="">`) y Links (`<a href="">`).
* **Demostración:** Insertar un texto, una imagen y un link en el `<body>` creado en la clase anterior.
* **Enlaces:** [C# Playground (DotNetFiddle)](https://dotnetfiddle.net/) | [Material de la Clase](starter-files/lesson-3-starter.md) ([Completado](completed-examples/lesson-3-completed.md), [Scripts](scripts/lesson-3-scripts.md))

## Clase 4: Introducción a CSS (10 minutos)
**Qué abarca:**
* ¿Qué es CSS y por qué separarlo de HTML? (Estructura vs. Presentación).
* Sintaxis básica: Selector, propiedad y valor (`selector { propiedad: valor; }`).
* Tipos de selectores básicos: Por etiqueta (ej. `p`), por clase (`.clase`) y por ID (`#id`).
* **Demostración:** Crear un archivo `.css` y vincularlo en el `<head>` del HTML con la etiqueta `<link>`.
* **Enlaces:** [C# Playground (DotNetFiddle)](https://dotnetfiddle.net/) | [Material de la Clase](starter-files/lesson-4-starter.md) ([Completado](completed-examples/lesson-4-completed.md), [Scripts](scripts/lesson-4-scripts.md))

## Clase 5: Estilo simple (10 minutos)
**Qué abarca:**
* Manipulación de colores: `color` (texto) y `background-color` (fondo).
* Tipografía y legibilidad: `font-family`, `font-size`.
* Modelo de caja express: Bordes (`border`), márgenes (`margin`) y cómo centrar un contenedor bloque con `margin: 0 auto;`.
* Alinear texto con `text-align: center;`.
* **Demostración:** Aplicar estilos básicos a los elementos creados en la Clase 3.
* **Enlaces:** [C# Playground (DotNetFiddle)](https://dotnetfiddle.net/) | [Material de la Clase](starter-files/lesson-5-starter.md) ([Completado](completed-examples/lesson-5-completed.md), [Scripts](scripts/lesson-5-scripts.md))

## Clase 6: Creación de una página web básica - HTML (10 minutos)
**Qué abarca:**
* Inicio del proyecto final: Una página de perfil personal o tarjeta de presentación (Profile Card).
* Maquetación en vivo: Se crea la estructura de la tarjeta usando un contenedor principal `<div>`.
* Contenido: Foto de perfil, nombre (`<h1>`), biografía corta (`<p>`) y una lista de habilidades o intereses (`<ul>`).
* **Meta:** Tener el HTML limpio, bien anidado y estructurado semánticamente.
* **Enlaces:** [C# Playground (DotNetFiddle)](https://dotnetfiddle.net/) | [Material de la Clase](starter-files/lesson-6-starter.md) ([Completado](completed-examples/lesson-6-completed.md), [Scripts](scripts/lesson-6-scripts.md))

## Clase 7: Creación de una página web básica - CSS (10 minutos)
**Qué abarca:**
* Estilización del proyecto final creado en la clase 6.
* Darle dimensiones a la tarjeta (ancho máximo) y centrarla en la pantalla.
* Añadir color de fondo general, bordes redondeados (`border-radius`) a la foto y a la tarjeta, y espaciado interno (`padding`) para que el texto respire.
* **Meta:** Transformar la estructura de la clase 6 en una interfaz atractiva.
* **Enlaces:** [C# Playground (DotNetFiddle)](https://dotnetfiddle.net/) | [Material de la Clase](starter-files/lesson-7-starter.md) ([Completado](completed-examples/lesson-7-completed.md), [Scripts](scripts/lesson-7-scripts.md))

## Clase 8: Recomendaciones y errores comunes para principiantes (6 minutos)
**Qué abarca:**
* **Errores:** Olvidar cerrar etiquetas, fallar en la ruta de las imágenes, escribir CSS sin vincular el archivo.
* **Buenas prácticas:** Escribir siempre en minúsculas (archivos y etiquetas), indentar el código para que sea legible, usar nombres descriptivos.
* **Cierre:** Dónde seguir aprendiendo (MDN Web Docs) y motivación final.
* **Enlaces:** [C# Playground (DotNetFiddle)](https://dotnetfiddle.net/) | [Material de la Clase](starter-files/lesson-8-starter.md) ([Completado](completed-examples/lesson-8-completed.md), [Scripts](scripts/lesson-8-scripts.md))
