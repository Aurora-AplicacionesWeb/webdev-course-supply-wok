# Clase 3 - Elementos HTML Comunes (Starter)

En esta clase aprenderemos a estructurar contenidos textuales, listas y elementos multimedia (imágenes y enlaces).

## Código Inicial

Utiliza este esqueleto HTML básico para empezar a agregar los elementos de la clase:

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Primera Página Web</title>
</head>
<body>

    <!-- TODO: 1. Agrega una jerarquía de textos (h1, h2, p) -->
    
    <!-- TODO: 2. Agrega una lista ordenada o desordenada (ul, ol, li) -->
    
    <!-- TODO: 3. Agrega una imagen (img) y un enlace (a) -->

</body>
</html>
```

## Instrucciones de Práctica
1. Agrega un título principal con `<h1>` que diga "Mi Perfil Profesional".
2. Escribe una breve descripción usando `<p>`.
3. Crea una lista desordenada (`<ul>`) con tres habilidades o pasatiempos.
4. Agrega un enlace hacia tu sitio favorito usando `<a href="...">` y una imagen descriptiva con `<img src="..." alt="...">`.
