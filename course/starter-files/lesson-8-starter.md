# Clase 8 - Recomendaciones y errores comunes (Starter)

A diferencia de las clases anteriores, esta lección no introduce código nuevo: revisa los 5 errores más frecuentes que comete un principiante al construir su primera página con HTML y CSS, usando como referencia la Profile Card creada en las Clases 6 y 7.

## Antes de empezar

Antes de ver la clase, revisa tu propio código de la Clase 7 (`lesson-7-completed.md`) y trata de identificar por tu cuenta:

1. ¿Cerraste todas tus etiquetas (`</p>`, `</div>`, `</li>`, etc.)?
2. ¿Tus nombres de archivo están en minúsculas y sin espacios?
3. ¿Tu `<link>` al CSS incluye `rel="stylesheet"`?
4. ¿Tu código está indentado o todo pegado en el margen izquierdo?
5. ¿Los nombres de tus clases CSS coinciden exactamente (mayúsculas/minúsculas) entre el HTML y el CSS?
