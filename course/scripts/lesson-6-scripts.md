# Lesson 6 - Script

Hola, bienvenidos a la lección número 6.

En esta clase vamos a comenzar nuestro proyecto final. Crearemos la estructura HTML de una tarjeta de presentación, también llamada *profile card*. En esta parte nos enfocaremos en organizar el contenido de la página. Todavía no trabajaremos el diseño visual; eso lo haremos en la siguiente lección con CSS.

Antes de escribir código, veamos qué contendrá nuestra tarjeta. Vamos a incluir una foto de perfil, un nombre, una pequeña descripción, una lista de intereses y un enlace. La idea es construir una página sencilla.

Ya estando aquí en la sección de HTML, vamos a escribir el contenido de nuestra página. Primero creamos un contenedor principal con un `div`. Este contenedor agrupará todo el contenido de la tarjeta. Para poder identificarlo más adelante, le agregamos una clase llamada `card`.

Dentro de este contenedor vamos a colocar una imagen de perfil con la etiqueta `img`. Esta etiqueta se usa para mostrar imágenes en una página web. Para insertar la imagen usamos el atributo `src`, donde colocamos el enlace de la imagen. En este caso voy a usar una imagen de ejemplo.

Debajo de la imagen escribimos el nombre usando la etiqueta `h1`. Yo pondré mi nombre. Usamos `h1` porque es una etiqueta para títulos principales y ayuda a que la estructura del contenido sea más fácil de entender. Además, el texto se ve más grande.

Ahora agregamos un nuevo párrafo con la etiqueta `p`. Aquí escribimos una presentación breve. Por ejemplo: “Hola, soy Zayd. Estoy aprendiendo a crear páginas web con HTML y CSS”.

Después de eso, añadimos un subtítulo con la etiqueta `h2`. Esta etiqueta se diferencia de `h1` porque se usa para subtítulos y se ve un poco más pequeña. En esta sección pondremos el título “Mis intereses”.

Debajo vamos a usar una lista con `ul`. Dentro de esta lista añadimos varios elementos `li`. Cada `li` representa un elemento de la lista. Por ejemplo, podemos poner intereses como diseño web, música, videojuegos y tecnología.

Las listas son útiles porque nos ayudan a mostrar información corta y ordenada. También se podrían usar para hobbies, habilidades o temas favoritos.

Ahora, después de eso, vamos a crear un enlace con la etiqueta `a`. En el atributo `href` podemos colocar cualquier enlace. En este caso voy a usar un enlace de ejemplo. También debemos escribir el texto que se mostrará en pantalla, por ejemplo: “Ver mi perfil”.

Si observamos el resultado, notaremos que la página aún se ve bastante simple. Eso es normal, porque HTML se encarga principalmente de la estructura y del contenido. Lo importante por ahora es que todo esté bien ordenado, bien escrito y correctamente anidado.

Antes de terminar, revisemos algunos consejos importantes. Asegúrate siempre de cerrar bien tus etiquetas, usar nombres de clases claros y mantener el código ordenado. También recuerda que el contenido debe ser fácil de leer.

En la siguiente clase usaremos CSS para transformar esta estructura en una tarjeta de presentación más atractiva, con colores, espacios, bordes redondeados y una mejor distribución visual.

Gracias por acompañarme en esta lección.
